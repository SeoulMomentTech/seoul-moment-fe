# 브랜드 이벤트 목록 관리 PRD

## 1. 개요
Admin 화면에서 제공되는 브랜드 프로모션 목록 관리 페이지입니다. 현재 운영 중인 브랜드 프로모션 리스트를 조회하고, 새로운 프로모션을 등록하거나 기존 프로모션을 수정/삭제할 수 있는 진입점을 제공합니다.

## 2. 페이지 구성 및 기능

### 2.1 이벤트 목록 페이지 (`/brand/promotion`)
- **리스트 노출**: 등록된 브랜드 프로모션 목록을 테이블 또는 카드 형태로 노출합니다. (UI는 '브랜드 관리 - 전체 브랜드' 페이지 참고)
- **표시 항목**: 브랜드명, 프로모션 프로필 이미지 등 (`GetBrandPromotionBrandListResponse` 규격 참조)
- **액션 기능**:
  - **등록 버튼**: 신규 브랜드 프로모션을 추가하는 등록 페이지(`/brand/promotion/add`)로 이동합니다.
  - **수정 버튼**: 개별 리스트 아이템 클릭 또는 수정 버튼 클릭 시 상세 수정 페이지(`/brand/promotion/edit/:id`)로 이동합니다.
  - **삭제 버튼**: 개별 리스트 요소를 제거합니다. (삭제 확인 모달 제공 권장)

## 3. API 연동 명세

### 3.1 목록 조회 (List)
- **훅 (Hook)**: `useBrandPromotionListQuery` (Web 클라이언트에서 사용하는 상단 브랜드 탭 조회 API와 동일한 규격 활용)
- **설명**: 전체 브랜드 프로모션 리스트를 패칭하여 화면에 노출합니다.
- **주요 데이터**: `id` (프로모션 ID), `brandId` (연결된 브랜드 ID), `name` (브랜드 이름), `profileImageUrl` (브랜드 썸네일)

### 3.2 삭제 (Delete)
- **훅 (Hook)**: `useDeleteBrandPromotionMutation`
- **설명**: 리스트에서 특정 브랜드 프로모션 항목을 서버에서 완전히 제거합니다.
- **동작**: API 호출 성공 시 `useBrandPromotionListQuery` 캐시를 무효화(Invalidate)하여 최신 리스트 상태로 뷰를 갱신합니다.
